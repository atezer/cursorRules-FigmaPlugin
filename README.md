# Cursor Rules Figma Plugin

## Proje Açıklaması
Bu proje, Cursor Rules için geliştirilmiş bir Figma eklentisidir.

## Kurulum
1. Gereksinimler:
   - Node.js
   - npm

2. Bağımlılıkları yükleyin:
\`\`\`bash
npm install
\`\`\`

## Geliştirme
Geliştirme modunda çalıştırmak için:
\`\`\`bash
npm run dev
\`\`\`

## Yapım
Projeyi derlemek için:
\`\`\`bash
npm run build
\`\`\`
